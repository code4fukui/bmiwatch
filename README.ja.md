# bmiwatch

体重とBMIを記録・管理するためのシンプルなクライアントサイドWebアプリです。Vanilla HTML、CSS、JavaScriptで構築されており、サーバーを必要とせず、すべてブラウザ上で動作します。

## ライブデモ

**https://code4fukui.github.io/bmiwatch/**

画面上には現在のBMI、標準体重の範囲、目標までの進捗が表示されます。また、グラフによって体重の推移を視覚的に確認できます。

## 機能

- **データの記録:** 体重を記録すると自動的にタイムスタンプが付与されます。身長は`localStorage`に保存されるため、次回以降の入力が簡単になります。
- **即時フィードバック:** 計算されたBMI、標準体重の範囲（BMI 18.5〜25.0に基づく）、およびその範囲に達するまでの差分を即座に確認できます。
- **推移の可視化:** 折れ線グラフで体重の推移を表示し、標準BMI範囲のガイドラインも併せて確認できます。
- **データ管理:** 記録のタイムスタンプの編集、個別のデータ削除、すべてのデータのクリアが可能です。
- **インポート / エクスポート:** CSVのインポートおよびエクスポート機能を使用して、データのバックアップと復元が簡単にできます。
- **PWA対応:** プログレッシブWebアプリ（PWA）としてホーム画面にインストールでき、オフラインファーストですばやくアクセス可能です。
- **使いやすい入力インターフェース:** ステッパーボタン（`+0.5`、`+0.1`など）を使用して、体重入力の微調整が簡単にできます。
- **レスポンシブデザイン:** デスクトップとモバイル端末の両方に適応する、クリーンでモダンなインターフェースです。

## 使い方

ブラウザで`index.html`を開くか、[ライブデモサイト](https://code4fukui.github.io/bmiwatch/)にアクセスしてください。

1. 体重と身長を入力し、「記録」ボタンを押します。
2. 身長はローカルに保存されるため、次回以降は体重を入力するだけで記録できます。
3. グラフと履歴テーブルで、現在の数値や推移を確認します。
4. 記録のタイムスタンプを編集するには、履歴テーブルの日時をクリックします。
5. ページ下部のボタンを使用して、すべての記録のインポート、エクスポート、または削除を行います。
