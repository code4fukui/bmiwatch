# bmiwatch

[English](README.md)

体重を記録し、身長と目標BMIから計算した目標体重へ近づいているかを確認するシンプルなWebアプリです。

公開ページ:
https://code4fukui.github.io/bmiwatch/

## 機能

- 体重を現在時刻つきで記録
- 身長と目標BMIを `localStorage` に保存
- 現在BMI、目標体重、目標までの差を表示
- 体重推移と目標体重のガイドラインをグラフ表示
- 記録日時の編集と記録削除
- CSVインポート / エクスポート

## 使い方

1. `体重 kg`、`身長 cm`、`目標BMI` を入力して `記録` を押します
2. 次回以降は保存済みの身長と目標BMIが自動で入ります
3. 履歴の日時を押すと記録時刻を編集できます
4. 下部の `CSVエクスポート` で記録を保存できます

## ファイル構成

- `index.html` - 画面構造
- `style.css` - スタイル
- `main.js` - 記録処理、BMI計算、グラフ描画、CSV入出力
