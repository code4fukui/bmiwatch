# dailysum

量と種別を記録するシンプルなWebアプリです。入力時刻つきで保存し、今日の合計量、日ごとのグラフ、直近のショートカット、CSVエクスポートを利用できます。

公開ページ:
https://code4fukui.github.io/dailysum/

## 機能

- 量と任意の種別を現在時刻つきで記録
- 今日の合計量を大きく表示
- 日単位に集計した種別ごとの積み上げ棒グラフを表示
- 直近5件のユニークな「量+種別」をショートカットボタン化
- 記録をCSVでエクスポート
- データを `localStorage` に保存

## 使い方

1. `量` を入力して `保存` を押します
2. よく使う量はショートカットボタンから即記録できます
3. 下部の `CSVエクスポート` で記録を保存できます
4. `全削除` ですべての記録を消せます

## ファイル構成

- `index.html` - 画面構造
- `style.css` - スタイル
- `main.js` - 記録処理、グラフ描画、CSV出力
